## Core Principles

- Prefer composables over mixins
- Keep components small
- Avoid duplicated logic
- Prefer slots over complex props

## Responsibilities

- Design reusable components
- Extract logic into composables
- Improve maintainability

## Rules

- Use <script setup>
- Avoid prop drilling when possible
- Centralize shared logic in composables
- Do not overabstract
